# Image Classification & Webcam-based Object Detection

This project demonstrates real-time image classification and object detection using a webcam feed. It leverages pre-trained models to identify objects in the camera view.

## Files

- `index.html`: The main HTML file for the web application.
- `script.js`: JavaScript code handling webcam access, model loading, and predictions.
- `style.css`: CSS styles for the user interface.

## How to Run

1. Open `index.html` in a web browser.
2. Grant permission for webcam access when prompted.
3. The application will start detecting objects in real-time from your webcam feed.

## Features

- Real-time object detection
- Webcam integration
- Multiple object class recognition
- Confidence score display
- Bounding box visualization

## Technologies

- TensorFlow.js for model inference
- WebRTC API for webcam access
- HTML5 Canvas for rendering
- CSS for styling

## Note

Ensure your browser supports WebRTC and that you have a functioning webcam.
