# Digit Recognition MNIST Dataset

This project implements a digit recognition system using the MNIST dataset. It uses machine learning to classify handwritten digits from 0 to 9.

## Files

- `index.html`: The main HTML file for the web application.
- `script.js`: JavaScript code containing the machine learning logic and UI interactions.
- `data.js`: Data handling and preprocessing for the MNIST dataset.

## How to Run

1. Open `index.html` in a web browser.
2. The application will load the MNIST dataset and train a model.
3. Draw a digit on the canvas and click "Recognize" to see the prediction.

## Features

- Interactive drawing canvas
- Real-time digit recognition
- Model training visualization
- Accuracy metrics display

## Technologies

- TensorFlow.js for machine learning
- HTML5 Canvas for drawing
- JavaScript for logic
