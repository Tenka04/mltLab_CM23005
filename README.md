## Getting Started

Each project is self-contained and can be run by opening the `index.html` file in a web browser. Some projects may require a local server for certain features.

## Prerequisites

- Modern web browser with JavaScript enabled
- Internet connection for loading TensorFlow.js libraries
- Webcam access for camera-based projects (if applicable)

## Technologies Used

- HTML5
- JavaScript (ES6+)
- TensorFlow.js
- CSS3
