# Text Sentiment Analysis

This project implements a text sentiment analysis application that classifies text as positive, negative, or neutral.

## Files

- `index.html`: The main HTML interface for the application.
- `script.js`: JavaScript code containing the sentiment analysis logic and model.

## How to Run

1. Open `index.html` in a web browser.
2. Enter text in the input field.
3. Click "Analyze" to see the sentiment prediction.

## Features

- Real-time text sentiment classification
- Confidence score display
- Support for multiple languages (if model supports)
- User-friendly web interface

## Technologies

- TensorFlow.js for natural language processing
- Pre-trained sentiment analysis model
- HTML5 and JavaScript for UI

## Model

Uses a pre-trained model for sentiment analysis, loaded via TensorFlow.js.
