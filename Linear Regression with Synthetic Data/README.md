# Linear Regression with Synthetic Data

This folder contains laboratory assignments focused on linear regression using synthetic data. Each assignment explores different aspects of linear regression implementation and analysis.

## Assignments

### [Lab Assignment 1](./Lab%20Assingment%201/)
Introduction to linear regression basics.

### [Lab Assignment 2](./Lab%20Assingment%202/)
Advanced linear regression techniques.

### [Lab Assignment 3](./Lab%20Assingment%203/)
Linear regression with multiple variables and evaluation.

## How to Run

Open the `index.html` file in each assignment folder in a web browser to view the interactive demonstrations.

## Topics Covered

- Data generation and visualization
- Model training and fitting
- Loss function optimization
- Model evaluation metrics
- Overfitting and underfitting analysis

## Technologies

- JavaScript
- HTML5 Canvas for plotting
- TensorFlow.js for computations
